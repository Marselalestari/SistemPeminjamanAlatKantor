### 🌐 Informasi Project
Aplikasi ini dikembangkan sebagai solusi digital untuk manajemen inventaris kantor, khusus untuk **Ujian Kompetensi Keahlian (UKK) Rekayasa Perangkat Lunak**.

* **🔗 Live Demo:** [sipak-sela.rf.gd](http://sipak-sela.rf.gd/)
* **☁️ Hosting:** InfinityFree
* **🛠️ Tech Stack:** PHP 8.3, Laravel, Tailwind CSS
