- ✔ Approval Peminjaman
- 📊 Dashboard Statistik
- 💰 Perhitungan Denda
- 🧾 Cetak Struk Peminjaman

## 🛠️ Tech Stack

- Laravel 12
- PHP 8.3
- MySQL
- Tailwind CSS

---

## ⚙️ Cara Install

```bash
git clone https://github.com/Marselalestari/SistemPeminjamanAlatKantor.git

cd SistemPeminjamanAlatKantor

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
